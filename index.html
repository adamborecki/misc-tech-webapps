Classroom Use Notes

Suggested student flow:

  1. Run Guided Tutorial first and clear all checkpoints.
  2. Run Quiz Mode, then retry for a higher score if needed.
  3. Open Canvas Deliverable, fill reflection fields, and copy the generated text.

These are plain static files, so hosting on GitHub Pages will work with no build step.
